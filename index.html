Automação

Assinatura de E-mail

Este é um projeto em PHP, HTML, CSS e JavaScript para facilitar a geração de assinaturas de e-mail. A aplicação permite que os usuários personalizem suas assinaturas correspondente para em e-mail.

Funcionalidades:
Personalização de informações, como nome, cargo, número de telefone, e-mail, escritório e estado. Pré-visualização em tempo real da assinatura gerada.

Calculadora de Economia

Descontos mensais e anual

A calculadora de desconto na conta de energia é uma ferramenta eficiente e intuitiva projetada para ajudar os usuários a calcular o valor do desconto mensal e anual em suas contas de energia elétrica.

Desenvolvida com HTML, CSS E JavaScript com foco na simplicidade e usabilidade, a calculadora visa proporcionar uma maneira fácil de estimar economias significativas em termos de custos de energia.
